Built and deployed 10 bare-metal projects on a Digilent Zybo (Zynq-7000) in a single Claude Code session — AXI GPIO, DMA, XADC, audio, Ethernet, SD/QSPI boot. The real value was debugging: tracing incomplete Vivado automation, Vitis 2024.2 interrupt ID encoding changes, and Bank 35 I/O conflicts.
